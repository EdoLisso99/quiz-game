# Quiz Game

## Progetto visibile sul link https://quiz-game-c5d87.web.app/

### Breve descrizione del progetto
Quiz Game è un sito web hostato con Firebase, che mostra all'utente un mini-quiz da risolvere. Sviluppato con React, è stato fatto più che altro per capire come funzionava la libreria Framer Motion (https://www.framer.com/motion/), la quale gestisce le animazioni dei vari componenti presenti sullo schermo.
